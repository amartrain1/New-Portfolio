Work

Featured: Movie Night

Movie Night is a movie database that can be used to search for specific movies using their title, or can be used to search for movies based off the preferred genre. Once you have selected a movie, you can click the 'Pick a Snack!' button to then select a snack based off the given search criteria. There were two API's used in the production of this project: Streaming Availability and Worldwide Recipes. Click on the image to the left to view the website!
